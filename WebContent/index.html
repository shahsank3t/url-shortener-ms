URL Shortener Microservice

To create a shorter url, append your url to:

https://url-shortener-ms-shahsank3t.c9users.io/new/

Example to create shorter url:

https://url-shortener-ms-shahsank3t.c9users.io/new/https://www.google.com
https://url-shortener-ms-shahsank3t.c9users.io/new/http://www.enginerd.in

Example of output:

{ "originalUrl": "https://www.google.com", "shorterUrl": "https://url-shortener-ms-shahsank3t.c9users.io/1" }
{ "originalUrl": "http://www.enginerd.in", "shorterUrl": "https://url-shortener-ms-shahsank3t.c9users.io/2" }

Usage:

https://url-shortener-ms-shahsank3t.c9users.io/1

Will redirect you to:

https://www.google.com
